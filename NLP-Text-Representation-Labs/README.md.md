# NLP Preprocessing Labs

## Project Overview

This folder includes my NLP preprocessing lab from my Applied AI coursework. The lab focused on preparing raw text data so it can be used by natural language processing models.

## Problem Statement

Raw text is messy and difficult for machine learning models to understand directly. Before using text in an AI model, it usually needs to be cleaned and processed. This lab explored common NLP preprocessing steps that help turn raw text into cleaner and more useful data.

## Tools and Technologies Used

- Python
- Google Colab / Jupyter Notebook
- NLTK
- spaCy
- Pandas
- Tokenization
- Stop word removal
- Lemmatization
- Text cleaning

## Methodology

The lab followed these main steps:

1. Loaded and reviewed sample text data
2. Tokenized text into smaller parts
3. Removed stop words and unnecessary punctuation
4. Applied lemmatization to reduce words to their base form
5. Compared preprocessing results
6. Reflected on why preprocessing is important in NLP

## Results

This lab showed how preprocessing can make text cleaner and easier for AI models to work with. It also showed that small preprocessing choices can change the final results of an NLP project.

## What I Learned

I learned that NLP is not just about running a model. A big part of the work is preparing the text first. Cleaning, tokenizing, and lemmatizing the data makes it easier for the model to understand patterns in the text.
