# Wildfire Data Science Project

## Project Overview

This project was created for my data science class and was based on the WiDS Datathon wildfire impact challenge. The goal was to use data science methods to understand wildfire impact and practice working with a real-world competition-style dataset.

## Problem Statement

Wildfires can affect communities, infrastructure, and the environment. This project focused on using data to make predictions and understand patterns related to wildfire impact.

## Tools and Technologies Used

- Python
- Google Colab / Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Kaggle
- Data cleaning
- Machine learning
- Model evaluation

## Methodology

The project followed these steps:

1. Registered for the Kaggle competition
2. Explored the dataset and understood the features
3. Cleaned and prepared the data
4. Trained machine learning models
5. Tested and improved the results
6. Submitted predictions to the Kaggle leaderboard
7. Documented the process with screenshots and reflections

## Results

This project helped me understand how data science competitions work and how models can be tested using public and private leaderboard scores. It also showed me that data preparation and model evaluation are very important parts of the process.

## What I Learned

I learned how to work with a real competition dataset and how to submit results on Kaggle. I also learned that improving a model takes testing, patience, and understanding the data instead of just running code once.
