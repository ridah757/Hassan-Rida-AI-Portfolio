# ITAI2373 NewsBot Project

## Project Overview

This project is a NewsBot AI project created for my Applied AI coursework. The goal was to build a news analysis system that can work with news data and use natural language processing techniques to understand and summarize information.

The project helped me practice working with text data, cleaning it, analyzing it, and using AI tools to generate useful insights.

## Problem Statement

Reading and analyzing a large amount of news articles manually can take a lot of time. This project shows how AI and NLP can help process news articles faster by identifying useful patterns, sentiment, and important information.

## Tools and Technologies Used

- Python
- Google Colab / Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Natural Language Processing
- Sentiment Analysis
- Data Visualization
- GitHub

## Methodology

The project followed these main steps:

1. Loaded and explored the news dataset
2. Cleaned and prepared the text data
3. Applied NLP techniques to analyze the articles
4. Used sentiment analysis and other text-processing methods
5. Created outputs and visualizations to explain the results
6. Documented the project in GitHub

## Results

The project showed how AI can be used to organize and analyze news content. It also helped demonstrate the importance of preprocessing, model selection, and clear documentation when building an AI project.

## What I Learned

From this project, I learned that NLP projects require more than just running a model. The text has to be cleaned, organized, and prepared correctly first. I also learned how important it is to explain the project clearly so someone else can understand what the code is doing.
